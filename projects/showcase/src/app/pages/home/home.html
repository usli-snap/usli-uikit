Design System

USLI UI SDK

A component library built with Angular for USLI applications. Consistent, accessible, and aligned with the USLI brand.

Browse Components Design Tokens

Features

●

Angular 21

Built with standalone components and signals. No NgModules required.

■

USLI Branded

Colors, typography, and spacing aligned with the USLI design language.

▲

Design Tokens

CSS custom properties for colors and typography — theme once, apply everywhere.

Quick Start

Import any component directly from the library.

import { UsliButtonComponent } from 'ui-sdk';

@Component({
  imports: [UsliButtonComponent],
  template: `<usli-button label="Save" [primary]="true" />`
})
export class MyComponent {}
