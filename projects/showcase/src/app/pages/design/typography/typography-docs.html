Design System / Typography

Typography

The USLI type scale uses Roboto and Open Sans from Google Fonts. 1rem = 16px throughout. All sizes and weights are available as CSS custom properties.

CSS Custom Properties

Reference these tokens in your component stylesheets.

/* Font families */
--font-roboto:     'Roboto', sans-serif;
--font-open-sans:  'Open Sans', sans-serif;

/* Font weights */
--fw-light:     300;   --fw-regular:   400;
--fw-medium:    500;   --fw-semibold:  600;
--fw-bold:      700;   --fw-black:     900;

/* Type scale */
--fs-h1: 3rem;        --fs-h2: 2.5rem;    --fs-h3: 2rem;
--fs-h4: 1.5rem;      --fs-h5: 1.25rem;   --fs-h6: 1rem;
--fs-body-1: 1rem;    --fs-body-2: 0.875rem;
--fs-caption: 0.75rem; --fs-overline: 0.625rem;
