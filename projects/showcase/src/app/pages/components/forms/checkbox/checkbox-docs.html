Components / Form Elements / Checkbox

Checkbox

A lightweight checkbox component for form inputs with support for reactive forms and form groups.

Validation Examples

The Checkbox component supports reactive forms validation.

Required Checkbox

You must agree to continue 

<label>
  <usli-checkbox [formControl]="formControl"></usli-checkbox>
  I agree to terms
</label>

Default Checked

<label>
  <usli-checkbox [formControl]="formControl"></usli-checkbox>
  Enable notifications
</label>

Disabled Checkbox

<label>
  <usli-checkbox [formControl]="formControl" [disabled]="true"></usli-checkbox>
  Subscribe to newsletter
</label>

Form Integration Patterns

Integrate checkboxes with Angular Reactive Forms for advanced form management.

Checkbox Group

Select options: 
<fieldset>
  <legend>Select options:</legend>
  <label>
    <usli-checkbox [formControl]="control"></usli-checkbox>
    Read
  </label>
  <label>
    <usli-checkbox [formControl]="control"></usli-checkbox>
    Write
  </label>
</fieldset>

Checkbox Group with FormArray

Select permissions: 
<label *ngFor="let control of controls; let i = index">
  <usli-checkbox [formControl]="control"></usli-checkbox>
  {{ permissions[i] }}
</label>

Conditional Visibility

Subscription options will appear here

<label>
  <usli-checkbox [formControl]="control"></usli-checkbox>
  Subscribe to newsletter
</label>

<div *ngIf="control.value">
  Subscription options...
</div>

Error States

Display validation errors and manage disabled states.

Required Field Error

This field is required 

<label>
  <usli-checkbox [formControl]="control"></usli-checkbox>
  I accept the terms
</label>
<p *ngIf="control.hasError('required')">
  This field is required
</p>

Disabled State

<label>
  <usli-checkbox [formControl]="control" [disabled]="true"></usli-checkbox>
  This checkbox is disabled
</label>

API

Inputs

Property Type Description
checked boolean The checked state of the checkbox
disabled boolean Whether the checkbox is disabled
value any The value of the checkbox

Outputs

Event Type Description
change EventEmitter Emitted when checkbox state changes
