Radio Group

A set of radio buttons for selecting a single option from multiple choices with reactive forms support.

Validation Examples

The Radio Group component supports reactive forms validation with required field checking.

Select an option (required):

Please select an option 

<usli-radio-group [formControl]="control">
  <legend>Select an option:</legend>
  <usli-radio
    *ngFor="let option of options"
    [value]="option.value"
    [label]="option.label"
  ></usli-radio>

Form Integration Patterns

Integrate radio groups with Angular Reactive Forms for advanced form management.

Radio Group in FormGroup

Make a selection: 
<form [formGroup]="form">
  <usli-radio-group [formControl]="control">
    <legend>Make a selection:</legend>
    <usli-radio
      *ngFor="let option of options"
      [value]="option.value"
      [label]="option.label"
    ></usli-radio>
  </usli-radio-group>
</form>

Error States

Display validation errors and manage disabled states effectively.

  This field is required
</p>

API

Inputs

Property Type Description
value any The value of the selected radio button
checked boolean Whether the radio button is selected
disabled boolean Whether the radio button is disabled
name string Name attribute for radio button grouping

Outputs

Event Type Description
change EventEmitter Emitted when radio selection changes
